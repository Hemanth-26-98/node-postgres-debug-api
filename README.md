# Node.js + PostgreSQL Debug API

## 🚀 Project Overview
This project demonstrates how database query performance affects API response time.

It compares:
- Slow query (without index)
- Optimized query (with index)

## 🧠 Key Learning
Database optimization directly improves backend performance.

# Node.js + PostgreSQL Debug API

## 🚀 Objective
Demonstrate backend API performance optimization using PostgreSQL.

## ⚙️ Features
- REST API using Express
- PostgreSQL database integration
- Slow vs optimized query comparison
- Indexing to improve performance

## 📌 Endpoints

### /slow-orders
- Fetches data without optimization
- Slower performance

### /fast-orders
- Uses indexing
- Improved performance

## 🛠️ Tech Stack
- Node.js
- Express
- PostgreSQL

## 🧠 Learnings
- API performance depends on database optimization
- Indexing improves response time
- Debugging backend performance issues
